## Format Reference

**Task syntax:** `- [STATE] <description> #tag [metadata...]`

| Marker | State |
|--------|-------|
| `[ ]` | Pending |
| `[/]` | In Progress |
| `[x]` | Completed |
| `[-]` | Cancelled |
| `[>]` | Deferred |
| `[?]` | Blocked |

**Required:**
- At least one `#tag` per task
- `_done:YYYY-MM-DD` when marking `[x]`
