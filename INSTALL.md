# Installation

## Prerequisites

- [Bun](https://bun.sh) v1.0 or later

```bash
curl -fsSL https://bun.sh/install | bash
```

## Quick Start

```bash
git clone https://github.com/0xsalt/tetris-demo.git
cd tetris-demo
bun install
bun start
```

Open `http://localhost:3000` in your browser.

## Custom Port

```bash
PORT=8080 bun start
```

## Development Mode

Auto-restarts on file changes:

```bash
bun dev
```
